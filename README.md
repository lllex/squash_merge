## Usage

```
usage: squash_merge.py [-h] branch1 branch2

merge with squash branch1 to branch2

positional arguments:
  branch1     branch that will be merged
  branch2     merge destination branch

optional arguments:
  -h, --help  show this help message and exit
```

Tested on

- ubuntu 16.04
- python 2.7.12
